# AURA-GDPR Frontend

A Vite + React + TypeScript + Tailwind starter for the AURA-GDPR frontend.

Quick start:

```bash
npm create vite@latest aura-gdpr-frontend -- --template react-ts
cd aura-gdpr-frontend
npm install
npm install axios react-router-dom
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p
npm run dev
```
